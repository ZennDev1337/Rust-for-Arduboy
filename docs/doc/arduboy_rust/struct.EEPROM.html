Expand description

This is the struct to store and read structs objects to/from eeprom memory.

Example

static e: EEPROM = EEPROM::new(10);
struct Scorebord {
    player1: u16,
    text: &'static str,
}
static mut s: Scorebord = Scorebord {
    player1: 0,
    text: "lol\0",
};

// init inside of the setup function
e.init(&mut s);
